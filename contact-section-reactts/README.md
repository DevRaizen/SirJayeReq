# Contact Section React + TS

This folder contains a React TypeScript + Tailwind version of the Figma contact section.

## Files

- `src/ContactSection.tsx` - Reusable typed component
- `src/App.tsx` - Example usage
- `src/main.tsx` - App entry point
- `src/styles.css` - Tailwind directives

## Notes

- The component currently uses Figma asset URLs for icons.
- For production, download the icons and replace the URL constants in `ContactSection.tsx`.
